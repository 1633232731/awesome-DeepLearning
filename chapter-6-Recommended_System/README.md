# movielens电影推荐系统

## 依赖模块

* paddlepaddle==2.0.0

## 项目介绍
```
6-1-Introduction_to_Recommended_System.ipynb 推荐系统介绍
6-2-Recommended System-Data_Processing.ipynb 推荐数据的预处理
6-3-Design_Movie_Recommended_System.ipynb 实现电影推荐系统
6-4-Movie_Recommended_System-Training_and_Extract_Feature.ipynb 电影推荐模型训练和特征提取
6-5-Movie_Recommended_System.ipynb 电影推荐
```
```
6-2-1-user_data.py 用户数据读取
6-2-2-movie_data.py 电影数据读取
6-2-3-ratings_data.py 评分数据读取
6-2-4-poster_data.py 海报数据读取
6-2-5-creat_data_reader.py 创建数据读取器
6-2-6-data_process_full_code.py 数据预处理完整代码
```

```
6-3-1-embedding.py embedding
6-3-2-id-embedding.py 用户idembedding
6-3-3-gender-embedding.py 性别embedding
6-3-4-age-embedding.py 年龄embedding
6-3-5-job-embedding.py 工作embedding
6-3-6-user-feature-merge.py 用户特征融合
6-3-7-movie-id-embedding.py  电影idembedding
6-3-8-movie-category-embedding.py  电影类别embedding
6-3-9-movie-title-embedding.py  电影标题embedding
6-3-10-movie-feature-merge.py   电影特征融合
6-3-11-similarity.py    相似度计算
6-3-12-movie-model.py  电影模型
```
```
6-4-1-training-evaluation.py  推荐模型的训练，评估，特征保存
```

```
6-5-1-read-feature.py 特征读取
6-5-2-calculate-similarity.py 计算用户和电影的相似度
6-5-3-movie-recommend.py  电影推荐
```
