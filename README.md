# 项目介绍

本项目将梳理1000+深度学习理论知识点及100+实践案例，内容涉及深度学习、计算机视觉、自然语言处理、强化学习、推荐系统等。

# 目录说明

* junior_class: 深度学习入门课程
* senior_class: 深度学习资深课程
* docs: 深度学习知识点建设

关于知识点更友好的阅读请参见[readthedocs](https://paddlepedia.readthedocs.io/)
